== Description ==

"Google Play Appbox" ermöglicht es, via Shortcode schnell und einfach Android-App-Details in Artikeln oder auf Seiten anzuzeigen. Eingebunden wird mit dem Shortcode [googleplay ID]. Dabei gibt es drei verschiedene Formatvorlagen:

- Simple: [googleplay ID] oder [googleplay simple ID]
- Banner [googleplay banner ID]
- Screenhot [googleplay screenshot ID]

Alternativ zur ID der Android-App kann auch die gesamte URL zum Play Store eingegeben werden.

== Frequently Asked Questions ==
* Wird das Plugin weiterentwickelt?
  Ja und nein. Es wird keine neuen Features geben, wohl aber Anpassungen und Fehlerkorrekturen. Auch an einer besseren Darstellung im RSS Feed wird gearbeitet (HTML, CSS).

* Kann man die Ausgabe anpassen?
  Ja - sämtliche Ausgabeelemente können mittels HTML und CSS nach belieben angepasst werden.
